## 概要

**HTTP** は、Webブラウザとサーバーが通信するためのアプリケーション層プロトコルです。
典型的にはTCP（HTTPSの場合はTLS over TCP）上で動作します。

## 要点

- **リクエスト・レスポンスモデル**: クライアントが要求し、サーバーが応答する
- **ステートレス**: リクエスト間で状態を保持しない（Cookieで状態を補う）
- **メソッド**: GET、POST、PUT、DELETE、PATCH など
- **ステータスコード**: 200系（成功）、300系（リダイレクト）、400系（クライアントエラー）、500系（サーバーエラー）

## 具体例

シンプルなGETリクエストの例:

```http
GET /hello HTTP/1.1
Host: example.com
User-Agent: personal-learning-site
Accept: */*
```

レスポンスの例:

```http
HTTP/1.1 200 OK
Content-Type: text/plain
Content-Length: 11

hello world
```
