## 概要

**型推論** は、明示的な型注釈がなくても、TypeScriptコンパイラが代入値や使用文脈から型を自動的に推測する機能です。

## 要点

- **代入による推論**: 変数の初期値から型を推論
- **戻り値の推論**: 関数の戻り値型を自動決定
- **構造的型付け**: 構造が一致すれば同じ型として扱う
- **`any` との違い**: 推論された型は安全に絞り込まれる

## 具体例

```ts
// 初期値から number が推論される
const count = 0;

// パラメータの文脈から string が推論される
const names = ["alice", "bob"];
names.map((n) => n.toUpperCase()); // n: string

// 戻り値は boolean に推論される
function isEven(n: number) {
  return n % 2 === 0;
}

// 判別可能なUnion
type Result =
  | { ok: true; value: number }
  | { ok: false; error: string };

function handle(r: Result) {
  if (r.ok) {
    // ここでは r.value が使える
    console.log(r.value.toFixed(2));
  } else {
    console.error(r.error);
  }
}
```
