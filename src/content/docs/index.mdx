## 学習履歴について

このサイトの学習履歴は、ブラウザの `localStorage` に保存されます。

- サーバーへ送信されません
- 別のブラウザ・別の端末には自動同期されません
- ローカル環境とGitHub Pagesでは保存領域が別になります
- 定期的に <a href="./settings/">設定画面</a> からエクスポートしておくことをおすすめします

詳細は <a href="./settings/">設定画面</a> または <a href="https://github.com/kf3225/de-glossary">README</a> を参照してください。
